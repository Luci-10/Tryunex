# Entry Marketplace

Entry is a deployable static marketplace site for new manufacturers. It includes:

- A polished storefront inspired by modern shopping marketplaces
- A seller onboarding form
- Product cards that update instantly in the browser
- Local persistence with `localStorage`

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deploy

You can deploy this folder directly to:

- Netlify
- Vercel (static deployment)
- GitHub Pages
- Cloudflare Pages

No build step is required.
